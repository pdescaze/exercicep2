Ce projet a pour objectif de mettre en pratique les notions acquises lors des cours précédents sur Git et Github à savoir:

	- Réaliser un commit
	- Lire l'historique
	- Se positionner sur un commit donnée
	- Découvrir les remotes et Github
	- Récupérer du code d'un autre repository
	- Créer son premier repository
	- Envoyer son code sur Github
	- Récupérer les modifications
